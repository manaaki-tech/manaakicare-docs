# Navigating the System

Learn how to move around Manaaki Central efficiently using the navigation menu, search, and other navigation features.

## Overview

Manaaki Central uses a consistent navigation structure to help you find what you need quickly:

- **Sidebar Menu** - Main navigation to different sections
- **Header Bar** - Theme toggle, refresh, and user profile
- **Search Bars** - Context-specific search within each section
- **Pagination** - Navigate through large lists of records

## Sidebar Menu

The left sidebar provides access to the main sections of Manaaki Central. The sidebar is always visible for quick navigation.

| Menu Item | Icon | Description |
|-----------|------|-------------|
| **<Term path="dashboard" />** | Home icon | Your personalized dashboard based on your role |
| **<Term path="serviceUser" plural />** | List icon | Search, view, and manage client records |
| **<Term path="referral" plural />** | Document icon | View all referrals with advanced filtering |
| **<Term path="serviceEpisode" plural />** | Activity icon | Track active service delivery episodes |

### Current Page Indicator

The currently active page is highlighted with a green/teal background in the sidebar, making it easy to see where you are in the system.

### Organisation Logo

The **Manaaki Central** logo at the top of the sidebar returns you to your dashboard when clicked.

## Header Bar

The header bar at the top of every page contains:

| Element | Description |
|---------|-------------|
| **Page Title** | Shows the current page name (e.g., "Intake Dashboard") |
| **Theme Toggle** | Sun/moon icon to switch between light and dark modes |
| **Refresh Button** | Circular arrow to reload the current page data |
| **User Avatar** | Your profile initial - click for account options |

## Search

Each section includes a context-specific search bar to help you find records quickly.

### How to Search

1. Click in the search field
2. Type your search term (client name, reference number, referrer name)
3. Results filter automatically as you type
4. Press **Enter** to confirm or click on a result

### Search Tips

- Use client names for quick lookups
- Reference numbers (e.g., REF-2026-DEFAULT-00047) provide exact matches
- Search works across multiple fields in each section

## Pagination

When viewing lists with many records, pagination controls appear at the bottom:

| Element | Description |
|---------|-------------|
| **Showing X-Y of Z** | Indicates which records are currently displayed |
| **Page X of Y** | Current page number and total pages |
| **< >** arrows | Navigate to previous/next page |

## Keyboard Shortcuts

Speed up navigation with keyboard shortcuts:

| Shortcut | Action |
|----------|--------|
| `Ctrl + F` | Focus search bar |
| `Enter` | Submit search |
| `Esc` | Close dialogs or cancel actions |
