# Navigating the System

Learn how to move around ManaakiCare efficiently using the navigation menu, search, and other navigation features.

:::note Work in Progress
This documentation is being developed. Check back soon for updates.
:::

## Overview

ManaakiCare uses a consistent navigation structure to help you find what you need quickly:

- **Sidebar Menu** - Main navigation to different sections
- **Search Bar** - Quick search for clients and records
- **Breadcrumbs** - Show your current location and path
- **Quick Actions** - Common tasks accessible from anywhere

## Sidebar Menu

The sidebar provides access to main sections:

- **Dashboard** - Your home screen
- **Clients** - Client search and management
- **Referrals** - Referral list and creation
- **Service Episodes** - Active service delivery tracking
- **Activities** - Case notes and interaction records
- **Reports** - Analytics and reporting (role-dependent)

### Collapsing the Sidebar

- Click the collapse icon to minimize the sidebar
- Hover to temporarily expand
- Click again to fully expand

## Search

The global search bar allows you to:

- Search for clients by name
- Find referrals by ID
- Locate specific records

### Search Tips

- Start typing to see suggestions
- Use quotes for exact matches
- Press Enter to see all results

## Breadcrumbs

Breadcrumbs show your current location:

```
Dashboard > Clients > John Smith > Service Episodes
```

- Click any breadcrumb to navigate back
- Helpful for understanding page hierarchy

## Keyboard Shortcuts

Speed up navigation with keyboard shortcuts:

| Shortcut | Action |
|----------|--------|
| `/` | Focus search bar |
| `g` then `d` | Go to Dashboard |
| `g` then `c` | Go to Clients |
| `g` then `r` | Go to Referrals |

## Mobile Navigation

On mobile devices:

- Tap the hamburger menu to open navigation
- Swipe from left edge to open sidebar
- Use bottom navigation for quick access
