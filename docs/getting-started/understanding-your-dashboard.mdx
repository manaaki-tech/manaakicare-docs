# Understanding Your Dashboard

The dashboard is your home screen in Manaaki Central, providing a personalized overview of your work and quick access to common tasks.

:::note Work in Progress
This documentation is being developed. Check back soon for updates.
:::

## Overview

Your dashboard displays information relevant to your role, including:

- Key statistics and metrics
- Recent activity
- Items requiring attention
- Quick action buttons

## Dashboard Components

### Statistics Cards

At the top of your dashboard, you'll see summary cards showing:

- Active clients
- Open referrals
- Pending tasks
- Recent activities

### Recent Activity

A timeline of recent actions, including:

- Client interactions
- Referral updates
- Document uploads
- Status changes

### Quick Actions

Buttons for common tasks:

- Create new referral
- Search clients
- View my caseload
- Record activity

## Role-Specific Views

Your dashboard adapts to your role:

| Role | Dashboard Focus |
|------|-----------------|
| Case Worker | Personal caseload, upcoming tasks, recent clients |
| Supervisor | Team overview, pending approvals, workload distribution |
| Program Manager | Program statistics, trends, performance metrics |

## Customizing Your View

- Use filters to focus on specific date ranges
- Toggle between different data views
- Collapse sections you don't need

## Refreshing Data

- Dashboard data updates automatically
- Click the refresh button to manually update
- Some statistics update at specific intervals
