# Case Worker Role

Understand the case worker role, including capabilities, responsibilities, and dashboard features.

:::note Work in Progress
This documentation is being developed. Check back soon for updates.
:::

## Overview

Case workers are front-line staff who manage direct client relationships and service delivery.

## Key Responsibilities

- Managing assigned caseload
- Creating and updating referrals
- Recording activities and case notes
- Maintaining client documentation
- Coordinating services
