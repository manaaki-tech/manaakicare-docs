## Colour Palette

| Colour | Hex | Usage |
|--------|-----|-------|
| **Sage** | `#5C8A72` | Primary brand, "Manaaki"/"Tiaki" text |
| **Teal** | `#4A8B8C` | Secondary accent, second word |
| **Amber** | `#C4A35A` | Highlights, focal points |
| **Stone** | `#3D3830` | Body text |
| **Cream** | `#FDFCFA` | Backgrounds |

---

## Quick Comparison

| Name | Symbol | Vibe |
|------|--------|------|
| **ManaakiPath** | Winding path + waypoints | Journey, progress, guidance |
| **ManaakiLink** | Chain links | Connection, relationship, network |
| **ManaakiCentral** | Hub + spokes | Coordination, central platform |
| **TiakiPath** | Shield + path | Protected journey, guardianship |
| **TiakiLink** | Shield + links | Secure connections, safe network |

## Feedback

Consider:
- Which name is easiest to say and remember?
- Does the symbol work at small sizes (favicon, app icon)?
- Does it feel right for NZ social services?
